Markup: HTML

  • paginile web conțin informație, în principal text
  • html este markup language, adică atașezi informație la fragmente de text, ca să spui ceva despre acele fragmente
  • exemple de markup: strong, a, ul/li, table, img, p, code, blockquote, form
  • block-level vs inline
  • div, span
  • a full web page; HEAD

Design vizual: CSS

  • inițial se punea informație vizuală direct în markup
  • css separă prezentarea de content
  • exemplu de stilizare de linkuri, paragrafe stilizat altfel link-urile dintr-un blockquote
  • clase și id-uri
  • selectori mai complecși

Interacțiune: JavaScript

  • inițial navigam din pagină în pagină prin linkuri și submit la forme, serverul controla orice schimbare în pagină
  • javascript - limbaj dinamic, fără legătură cu java, cu niște quirks
  • api dom: elemente, fereastră, url, event-uri
  • biblioteci - jquery
  • exemple de selectoare cu jquery în web inspector
  • toggle o clasă programatic

  • browserele implementează diverse feature-uri în ritmul lor; bug-uri; userii au legacy browsers
  • progressive enhancement, graceful degradation
  • browser detection vs. feature detection
